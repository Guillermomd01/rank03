# 42 Exam Rank 03 – New Exercises

These are the **new python exercises** for the **42 Exam Rank 03** 2026.  

> Note: The subjects may not be exactly the same as the real exam, but they are very close.  

> It’s difficult to remember all subjects and code at the same time, so these exercises are here to help you practice.  

> Subject's may not be the same as exam but exercises are !

> ⚠️ Note on “authorized functions”: In the original subjects, it sometimes says **None** for authorized functions, but for these exercises, **you are allowed to use any functions** to solve them.  

## Structure

- There are **6 exercises in total**.  
- **5 exercises** are worth **16 points** each.  
- The **last exercise** is worth **20 points**.  

## Exercises

| Exercise | Description | Points |
|----------|-------------|--------|
| `ex00/bracket_validator` | Check if brackets in a string are correctly matched | 16 |
| `ex01/echo_validator` | Validate and print echoed input | 16 |
| `ex01/matrix_reverse` | Reverse a given matrix | 16 |
| `ex02/py_pattern_tracker` | Count numeric pairs in a string | 16 |
| `ex03/py_shadow_merge` & `py_string_permutation_checker` | Merge shadow strings / Check if two strings are permutations | 16 |
| `ex04/py_string_sculptor` & `twist_sequence` | Sculpt strings by index case alternation / Twist sequences | 16 |
| `ex05/whisper_cipher` | Encode/decode strings using a cipher | 20 |

> Note: Two of us were doing the exam at the same time.  
> We did not have exactly the same subjects, but we covered about **80% of the exam** together.
